# depp-link-generator
App deeplinks tool. 

**Prerequisites**
1. Login to your BOB account
2. Use the deep link generator

**Installing**
1. Download link-generator.html
2. Open the HTML file

**How to use**
1. Input the URL
2. Choose the Region
3. Click Generate
4. Copy the "deeplink" attribute of the JSON generated
